# 2维安卓

Unity 2022.3.62 2D action prototype.

玩家通过键盘或虚拟摇杆移动、翻滚躲避敌人，使用散射子弹、近战和环形弹幕击败不断刷新的敌人，收集掉落金币并坚持生存。

## Project

- Unity version: 2022.3.62
- Main scene: `Assets/Scenes/MainScene.unity`
- Core scripts: `Assets/Scripts/PlayerMovement.cs`, `Assets/Scripts/PlayerCombat.cs`, `Assets/Scripts/Enemy.cs`, `Assets/Scripts/GameManager.cs`
